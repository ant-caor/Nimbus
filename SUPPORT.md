# Support

Thanks for using Nimbus. Here is where to go depending on what you need.

## Questions and help

- **How-to questions / usage help:** open a [GitHub Discussion](https://github.com/ant-caor/nimbus/discussions)
  (or a question issue if Discussions are not enabled).
- **Documentation:** start with the [README](README.md) and the design write-up
  in [DESIGN.md](DESIGN.md). API reference lives on
  [pkg.go.dev](https://pkg.go.dev/github.com/ant-caor/nimbus).

## Bugs and feature requests

- **Bugs:** open a [bug report](https://github.com/ant-caor/nimbus/issues/new?template=bug_report.yml).
- **Feature requests:** open a [feature request](https://github.com/ant-caor/nimbus/issues/new?template=feature_request.yml).

Please search existing issues first to avoid duplicates.

## Security

**Do not** report security vulnerabilities through public issues. Follow the
private process in [SECURITY.md](SECURITY.md).

## Expectations

Nimbus is maintained on a best-effort basis. There is no commercial support or
SLA. Well-described issues and pull requests get attention fastest.
